# Parcel Starter Kit

> Simple boilerplate: Parcel, Babel, Autoprefixer, Posthtml, Eslint and SCSS

## Build Setup

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:1234
yarn serve

# lint code
yarn lint

# build for production with minification
yarn build
```
